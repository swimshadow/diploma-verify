# micro-swagga-label (фрагмент в репозитории)

Здесь лежит только **`docs_service`** — агрегатор OpenAPI из проекта **micro-swagga-label**: через Docker Socket находит контейнеры с label `docs.route` и собирает единый Swagger UI (`http://localhost/docs`).

При обновлении агрегатора можно скопировать заново каталог `docs_service` из апстрима.
